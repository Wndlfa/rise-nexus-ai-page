# B2 Nexus – Landing Page

Landing page oficial do B2 Nexus, plataforma SaaS de automação de atendimento com IA, CRM, disparos em massa e integrações com WhatsApp, Instagram, Facebook Leads e Google Calendar.

Este repositório contém **apenas** o código da landing page (frontend estático), sem lógica interna do painel SaaS.

---

## Tecnologias

- Vite
- React + TypeScript
- Tailwind CSS
- Shadcn UI
- Deploy em Vercel

---

## Como rodar o projeto localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/Wndlfa/landing-page-b2nexus.git
   cd landing-page-b2nexus

2. Instale as dependências:

   ```bash
   npm install

3. Inicie o servidor de desenvolvimento:

   ```bash
   npm run dev

4. Abra [http://localhost:8080](http://localhost:8080) no seu navegador para ver o resultado.

5. Contribuindo para o projeto:

   - Faça um fork do repositório.
   - Crie uma branch para sua feature ou correção: `git checkout -b feature/feature-name`.
   - Faça suas alterações e comite-as: `git commit -m "Descrição da alteração"`.
   - Envie sua branch para o repositório remoto: `git push origin feature/feature-name`.
   - Abra um Pull Request no GitHub.
